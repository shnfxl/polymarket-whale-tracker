# Security Policy

## Reporting a Vulnerability
Please do not open public issues for sensitive vulnerabilities.

Report security issues privately to the maintainer with:
- Affected component/file
- Reproduction steps
- Impact assessment
- Suggested remediation (if available)

## Secrets and Credentials
- Never commit `.env` files, API keys, or bot tokens.
- Rotate Telegram bot credentials immediately if exposed.

## Scope
This repository is an alerting/monitoring bot and does not execute trades.
